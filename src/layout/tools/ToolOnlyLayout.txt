0. CONCEPTUAL PURPOSE
---------------------
This file defines **UI strings / labels / microcopy** for pages that show
a single SSA tool in **TOOL MODE**.

Examples:
- A single calculator or planner (meals, storehouse, garden bed, flock, etc.)
- A focused flow (one preservation run, one cleaning cycle, one feast plan)

The layout must:

- Present each tool as a **household flow or rhythm**, not a generic module.
- Use language tied to **cycles, seasons, storehouse, yield, provision,
  feast, rhythm, householder, flow**.
- Work alone (TOOL MODE) while hinting how it can connect into
  STEWARDSHIP MODE later.
- Stay clear and friendly, not cryptic or preachy.
