0. CONCEPTUAL PURPOSE
---------------------
This file defines **UI strings / labels / microcopy** for the Stewardship
Dashboard layout.

The layout must:

- Present SSA as a **household steward**, not a productivity app.
- Speak in terms of **cycles, seasons, storehouse, yields, provisions,
  rhythms, feasts, householder, flows**.
- Work in:
  - TOOL MODE: a single-domain user just using one planner or calculator.
  - STEWARDSHIP MODE: a householder seeing cross-domain patterns.

It is **not** code. It is a copy + label reference so React components can
pull from a consistent “web of meaning” vocabulary later.
